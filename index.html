JavaScript

Introdução

O que é?

JavaScript é a linguagem nativa do navegador, sendo a linguagem mais popular de programação e também de fácil aprendizado, utilizada principalmente para o desenvolvimento web e desenvolvimento de software, podendo ser usada tanto no front-end e no back-end.

Características

É uma linguagem leve, multiplataforma, orientada a objeto, tem uma fácil conexão com o html e css utilizando-se do DOM (Document Object Model).

JavaScript X Java

Mesmo tendo nomes parecidos são linguagens completamente diferentes, possuindo sintaxe, semântica e usos diferentes. O JavaScript teve vários nomes, o primeiro foi Mocha, o segundo foi LiveScript, e por conta da enorme fama do Java mudou seu nome para JavaScript, como um jogo de marketing.

História

O JavaScript foi criado em 1995 pelo Brendan Eich. Seu primeiro nome foi Mocha e, logo depois passou a ser chamado de LiveScript, e por último ficou conhecido como JavaScript. O primeiro navegador que teve suporte ao JavaScript foi o Netscape, com ele a internet pôde ficar mais dinâmica, tendo mais interação e animações na página.

Em 1996, a Netscape começou a trabalhar no ECMA International que visava tornar o JavaScript um padrão do setor. Desde então, o JavaScript padronizado é chamado de ECMAScript e especificado sob a ECMA-262, cuja última edição (décima primeira, ES2020) está disponível desde junho de 2020.

A popularidade do JavaScript aumentou ainda mais com o surgimento do Node.js, o interpretador multiplataforma mais popular de ambiente de execução JavaScript fora do navegador. Node.js foi criado utilizando o V8 JavaScript Engine permitindo que os desenvolvedores utilizem o JavaScript como linguagem de script para automatizar as coisas em um PC e criar servidores HTTP e WebSocket totalmente funcionais.

Front-End

Entre as incontáveis funções do JavaScript no front-end, as principais são fazer as interações e as animações do site, muitas dessas interações são feitas pelo uso do DOM (Document Object Model). Como o JavaScript já é uma linguagem nativa dos navegadores não é necessário nenhuma ferramenta para a execução de seus códigos na parte do front-end.

Para facilitar um pouco o desenvolvimento de sites existem alguns frameworks voltados ao uso do HTML, CSS e JavaScript

Frameworks:

  • React
  • Angular
  • AngularJS
  • Vue.js
  • Bootstrap

Back-End

No back-end o JavaScript tem o mesmo papel de outras linguagens de programação back-end, como, fazer a integração, gerenciamento de dados, tratamento das informações recebidas dos formulários, e entre outros.

Para o JavaScript poder atuar no back-end é necessário o uso do Node.js para a execução de seus códigos fora de um navegador Web, sendo ele um software de código aberto, multiplataforma, baseado no interpretador V8 do Google.

Hello World

O "Hello World" é um exemplo clássico usado em qualquer linguagem para demonstrar como é feito um código que mostra um "Hello World" em seu console ou sua tela.

Por exemplo, ficaria assim um código no JavaScript que mostra um "Hello World" em seu site:

<script>
window.alert("Hello World!")
</script>

A tag <script> é uma tag HTML e é necessária para a execução do código.

No trecho window.alert("Hello World!") o comando window se refere a janela do navegador, alert() é uma função que executa um alerta com a mensagem que está dentro do parentêses.
